Preprocessor είναι η κλάση η οποία διαμορφώνει τα δεδομένα που κρατούνται στην μνήμη του DC για κατάλληλη χρήση στα
μοντέλα. Παρέχει μετατροπές, κανονικοποιήσεις, αυτοματοποιημένες κατασκευές πρόσθετων στηλών (dummy variables, cos/sin
variables κλπ)

Είσοδος
-------
EF                              To στιγμιότυπο του EnergyForcaster στο οποίο ανήκει
DC_list     (dict)              Το λεξικό με τα αρχεία τα οποία είναι ανοιχτά στον DC.

Χαρακτηριστικά
--------------
_EF                             Ο EnergyForecaster στον οποίο ανήκει


Συναρτήσεις
-----------
boxcox							Μετροπή Box-Cox
croston_method					Μέθοδος Croston για αραιές σειρές
day_hour						Δείκτης/One-hot/Κυκλική μεταβλητή για ώρες ημέρας
differentiate					Διαφοροποίηση κανονική/εποχική
downgrade_data_frequency		Κατεβάζει την συχνότητα των δεδομένων με χρήση κάποιας δοθείσας συνάρτησης
exp								Εκθετική μετατροπή φυσική
exp2							Εκθετική μετατροπή δυαδική
fill_backward					Πλήρωση κενών τιμών προς τα πίσω
fill_forward					Πλήρωση κενών τιμών προς τα εμπρός
fill_linear						Πλήρωση κενών τιμών προς με γραμμική παρεμβολή
lagged_series					Δημιουργία υστερημένων χρονοσειρών
limit_output					Μετατροπή εξόδου για περιορισμό τιμών σε διάστημα
ln								Λογαριθμική μετατροπή φυσική
log								Λογαριθμική μετατροπή με επιλογή βάσης
log10							Λογαριθμική μετατροπή δεκαδική
log2							Λογαριθμική μετατροπή δυαδική
minmax							Κανονικοποίηση Min-Max
month_weekdays					Αριθμός ημερών εβδομάδας ανά μήνα
monthday						Δείκτης/One-hot/Κυκλική μεταβλητή για ημέρες μήνα
public_holidays					One-hot μεταβλητή για εθνικές αργίες
reverse_trans                   Αντίστροφη εφαρμογή λίστας μετασχηματισμών σε δεδομένα
robust							Κανονικοποίηση Robust
standard						Κανονικοποίηση Standard
to_timestamp					Μετατροπή string σε timestamp
weekday							Δείκτης/One-hot/Κυκλική μεταβλητή για ημέρες εβδομάδας
weekend							One-hot μεταβλητή για καθημερινές/Σαββατοκύριακα
year_day						Δείκτης/One-hot/Κυκλική μεταβλητή για ημέρες έτους
year_month						Δείκτης/One-hot/Κυκλική μεταβλητή για μήνα έτους
year_week						Δείκτης/One-hot/Κυκλική μεταβλητή για εβδομάδα έτους


Βοηθητικές συναρτήσεις
----------------------
_calculate_year_day				Υπολογίζει την ημέρα του χρόνου.
_cos_sin_transformation			Δημιουργεί από δείκτες, κυκλικές σειρές ημιτόνου/συνημιτόνου.

_count_month_weekdays			Υπολογίζει τον αριθμό των ημερών εβδομάδας για κάποιον μήνα.
_create_dummies_array			Δημιουργεί από δείκτες, one-hot πίνακες.
_create_indicators				Δημιουργεί τις διάφορες χρονικές μεταβλητές/δείκτες (πχ weekday),
								ελέγχοντας παράλληλα την ορθή είσοδο ζώνης ώρας.
_date_mask						Δημιουργεί μία χρονοσειρά boolean για επιλογή δεδομένων μεταξύ ημερομηνιών.
_is_in_dates					Ελέγχει αν η δοσμένη ημερομηνία είναι σε αυτές της λίστας των
								εθνικών αργιών
_time_zone						Επιστρέφει το pytz.timezone για ένα όνομα timezone.
_timestamps_to_dates			Επιστρέφει είσοδο σειράς timestamp σε ημερομηνίες μορφής datetime.datetime.
